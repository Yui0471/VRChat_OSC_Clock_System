# VRChat OSC Clock System

本データはVRChat想定のアバターにセットアップし、OSC送信を介してPCのローカル時間をアバターに表示させるものです。
時刻表示の他、汎用的なデジタルディスプレイとしても使用できます。(タイマーやカウントダウンなど)

# Requirement

* Windows 10 以降想定


## Pythonを使用する場合

* Python 3.8以上
* python-osc 1.8.0

```bash
pip install python-osc
```
